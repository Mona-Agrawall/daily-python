# 🐍 daily-python

> A personal Python practice journal — one script at a time.

---

## 📖 About

**daily-python** is a growing collection of Python scripts built through consistent daily practice. Each file explores a core concept — from array manipulation and sorting algorithms to pandas data wrangling — making this repo both a learning log and a quick reference.

---

## 📂 What's Inside

| File | Topic |
|------|-------|
| `array.py` | Array creation and manipulation |
| `indexing.py` | Indexing and slicing techniques |
| `iterating.py` | Looping and iteration patterns |
| `concatenation.py` | Joining and merging sequences |
| `copy.py` | Shallow vs. deep copy |
| `sort.py` | Built-in sorting methods |
| `selectionSort.py` | Selection sort algorithm from scratch |
| `pandas/loc.py` | Label-based selection with `.loc` in pandas |

---

## 🚀 Getting Started

**Clone the repo:**
```bash
git clone https://github.com/Mona-Agrawall/daily-python.git
cd daily-python
```

**Install dependencies (for pandas scripts):**
```bash
pip install pandas
```

**Run any script:**
```bash
python array.py
python pandas/loc.py
```
